# 時鐘Plus（Clock Plus）

時鐘Plus是一款使用 TurboWarp 開發的高自由度時鐘程式。

本專案以「時鐘」為核心，提供高度自訂化的介面、多功能設定以及模組化設計，讓使用者可以依照自己的需求打造專屬的時鐘。

---

## ✨ 主要特色

### 🕒 多樣化時鐘功能

- 數位時鐘顯示
- 多種時間格式
- 日期與星期顯示
- 可自由調整外觀

### 🧩 高自由度資訊欄

可自由新增、移除與排列資訊欄內容，例如：

- 日期
- 星期
- 電量
- 網路資訊
- 天氣（未來規劃）
- 自訂文字
- 更多可擴充資訊

### ⚙️ 多功能設定

提供完整的自訂功能，包括：

- 顏色
- 字型
- 字體大小
- 透明度
- 排版
- 顯示方式
- 更新設定

### 🚀 高效能架構

本專案採用資料驅動（Data-driven）設計：

- JSON 資料管理
- Renderer 渲染架構
- 模組化設計
- 易於維護與擴充

### 🔄 線上更新

程式可透過 GitHub 線上檢查最新版本資訊，提醒使用者下載最新版程式。
